Meu primeiro formulário

Todo formulário tem que estar entre as tags <form></form>

Se o autopreenchimento estiver incomodando, dentro da tag form, digite autocomplete="off"

É necessário criar uma ligação entre a caixa de texto e o nome que a identifica. Para isso, usa-se o label. Selecione "Nome" e "Sobrenome", envelope com label, depois coloque o id dentro do campo "for". O id deve ser estabelecido como no código aqui feito.

O método post, dentro de form, serve para não enviar os dados do formulário para a url. Quando os dados não forem sensíveis, pode-se utilizar o método get. Utiliza-se o get, também, quando for compartilhar um link com uma busca específica. Passou de 3000 bytes, não dá para usar o get.
